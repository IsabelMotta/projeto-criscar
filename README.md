# Projeto CrisCar Autos

Este repositório contém o código-fonte de uma landing page automotiva desenvolvida para fins de portfólio profissional. O projeto foca na fidelidade visual entre o requisito de design (esboço) e a implementação final.

## Objetivos Técnicos

* **Parametrização de Layout**: Ajuste fino de elementos CSS para integração entre o bloco de identidade visual (logo) e o menu de navegação.
* **Responsividade**: Implementação de media queries para adaptação automática do conteúdo em dispositivos móveis, corrigindo falhas de escala no menu e na grade de exibição.
* **Estrutura de Componentes**: Organização de uma grade de produtos utilizando CSS Grid com integração de links dinâmicos para atendimento via WhatsApp.

## Tecnologias Utilizadas

* HTML5
* CSS3 (Flexbox, Grid, Clip-path, Media Queries)
* Node.js (Ambiente para execução do servidor local)

## Estrutura do Projeto

* `index.html`: Arquivo principal contendo estrutura e estilização.
* `server.cjs`: Script do servidor para gestão do ambiente local.
* `*.jfif`: Ativos de imagem utilizados no layout.
